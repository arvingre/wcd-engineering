# Engineering Memory Templates

Use these templates when creating durable Engineering Memory records. Replace placeholder identifiers and fields; do not remove required metadata without updating `architecture/engineering-memory.md`.

Available templates:

- `decision.md`
- `incident.md`
- `runbook.md`
- `lesson.md`
- `pattern.md`
- `project-state.md`

A template creates a Draft record, not an Active record. Promotion requires verification and an accepted trigger.
